# 陀螺仪直线调试操作手册

## 前置

烧录后上电，等待启动完成（串口出现 `Dual Motor Control Ready` + MPU6500 初始化成功）。

## 调试流程（全程串口命令，不改代码）

### 第一步：切陀螺仪模式 + 速率 PID 初测

```
mode 1                  → 强制陀螺仪模式 B (不用灰度)
st 0.5 0.15             → 直行 0.5m @ 0.15m/s
```

观察：车是否偏航？偏了说明速率 PID 不够。

### 第二步：调速率 PID (Gp / Gi)

```
Gp 0.8                  → 增大速率 P
Gi 0.05                 → 加 I
st 0.5 0.15             → 再测
```

反复直到车身无明显偏航。

> `Gp` 越大，纠偏越猛（太大车会左右摆）。
> `Gi` 补偿微小的持续偏航。

### 第三步：开航向锁定

速率 PID 调好后，开启航向锁定主动保持方向：

```
lock 1                  → 开航向锁定
Gh 2.0                  → 航向角度 P
Gi_h 0.05               → 航向角度 I
st 1.0 0.2              → 更长距离验证
```

> `Gh` 控制偏离原方向后拉回的力度，越大越猛。

### 第四步：长距离验证

```
st 2.0 0.3              → 2m @ 0.3m/s
```

若偏航：继续微调 `Gh`、`Gi_h`。若稳定：校准完成。

## 随时查看

| 命令 | 输出 |
|------|------|
| `diag` | `YAW=±x.x CORR=±x.xxx HEAD=±x.xxx LOCK=1` |
| `imu` | Roll / Pitch / Yaw (°) + YawRate (°/s) |
| `gs` | 灰度传感器 8bit |
| `?` | 电机状态 + 帮助 |

## 参数含义

| 参数 | 默认 | 作用 |
|------|------|------|
| `Gp` | 0.5 | 每 °/s 偏航速率 → 多少 m/s 转向修正 |
| `Gi` | 0.02 | 累积静差补偿 |
| `lock 1` | — | 锁定当前航向角，偏了主动拉回 |
| `Gh` | 2.0 | 每 rad 航向角偏差 → 多少 m/s 修正 |
| `Gi_h` | 0.05 | 缓慢补偿零漂 |

## 回退到循线模式

```
mode 0                  → 切回模式 A (灰度循线)
```
